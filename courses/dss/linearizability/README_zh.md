# Linearizability（线性化一致性 测试工具）

## 目标

`linearizability` 提供用于验证分布式键值存储实现（如 `kvraft`）是否满足线性化（linearizability）一致性的工具与模型。它包含模型实现、解析与测试数据，便于自动化检验客户端 API 的线性化性质。

## 主要内容
- 模型实现：`src/model.rs`、`src/models.rs` 中包含用于建模与验证的代码。
- 测试数据：`test_data/` 下包含若干操作序列（如 `c01-ok.txt`, `c01-bad.txt`），用于验证实现是否线性化或触发错误场景。

## 如何使用
- 阅读 `test_data` 中的示例文件，理解操作序列格式。
- 使用代码中的模型与工具对 `kvraft` 的运行日志或模拟执行序列进行验证，以检查是否满足线性化定义。

## 注意
- 该模块主要作为测试/验证工具，不依赖网络或 RPC 层；在调试 `kvraft` 时非常有用。
